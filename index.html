pyAPRS

pyAPRS (by the lack of a good name) is a small client for APRS. It's written in python using the GTK toolkit (pygtk) and currently supports link to openstreetmap maps. It aims at providing a simple to use, fast and lightweight node to the APRS network. It's not a replacement for other applications (like link to Xastir) but to complement them.

Features

  • Fast map scrolling and zooming
  • Openstreetmap local cache for off-line use
  • Full RF interfaces for TNC and soundmodem devices
